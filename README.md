# TetrisQuest

クエストベースのひねりを加えた、クラシックなテトリスゲームです。

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://dma-cmyk.github.io/TetrisQuest/)

## 🌟 特徴

*   クラシックなテトリスのゲームプレイ
*   ユニークなチャレンジがあるクエストシステム
*   スコア追跡とリーダーボード
*   複数の難易度レベル

## 🎮 遊び方

1.  **ピースの移動:** 矢印キーを使って、落ちてくるテトロミノを動かします。
2.  **回転:** 上矢印キーでピースを回転させます。
3.  **ドロップ:** スペースバーを押すと、ピースが一瞬で下に落ちます。
4.  **ラインを揃える:** 横一列を埋めると、そのラインが消えてスコアが加算されます。
5.  **クエストを達成する:** クエストログで特別なチャレンジを確認し、追加ポイントを獲得しましょう。

## 🚀 インストール

TetrisQuestをプレイするには、お使いのウェブブラウザで `index.html` ファイルを開くだけです。

```bash
# リポジトリをクローンする (任意)
git clone https://github.com/dma-cmyk/TetrisQuest.git
cd TetrisQuest
# ブラウザで index.html を開く
```
